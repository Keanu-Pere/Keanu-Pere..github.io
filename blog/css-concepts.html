What is the difference between Margin, Border, and Padding?

Padding

  • Padding is how much space is within an object sides and it’s contents.
  • Padding sets a spacing between an attributes outer box and its parent or between its neighbor.

Margin

  • Margin sets spacing inside a parents box for an attribute as a child.
  • Margin is how much space is around an object.

Border

  • Border is between the margin and padding of an object. Only counted when given an actual width otherwise, it’s zero.
  • Border places a border spacing fully around inside a parent or child inner attributes.
